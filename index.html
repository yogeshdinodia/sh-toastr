Usage:

To use this library, simply you have to link CSS (sh_toaster.css) and JS Script (sh_toaster.js)

Then you have to initialize a simple Toaster on button click in JS script:

document.addEventListener('DOMContentLoaded', () => {
   const showSuccessButton = document.getElementById('showProgSuccess');
   showSuccessButton.addEventListener('click', () => {
   // Toast.success (supports: success, danger, warning, info) //
    Toast.success('Success! This is a success message.');
  });
});


Want to change setting:

document.addEventListener('DOMContentLoaded', () => {
   const showSuccessButton = document.getElementById('showProgSuccess');
   showSuccessButton.addEventListener('click', () => {
   let Settings = {
      duration: 2000,
      showProgress: true,
      toastLocation: 'bottom'
   };
   Toast.success('Success! This is a success message.', Settings);
   });
});
